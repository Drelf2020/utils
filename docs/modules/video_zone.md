## def get_zone_info_by_tid()

| name | type | description |
| ---- | ---- | ----------- |
| tid  | int  | 分区的 tid  |

根据 tid 获取信息。

**Returns:** `Tuple[dict | None, dict | None]`: 第一个是主分区，第二个是子分区，没有时返回 None。

## def get_zone_info_by_name()

| name | type | description  |

| name | str  | 分区的名称。 |

**Returns:** `Tuple[dict | None, dict | None]`: 第一个是主分区，第二个是子分区，没有时返回 None。
